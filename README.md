# Art Of Motion — Fitness Platform

A full-stack web application for the **Art Of Motion** fitness brand. The platform combines a marketing landing page for various training disciplines (Strength, Stretching, MFR, Gym, Balance Boards) with an interactive scheduling, client booking system, and trainer admin CRM.

## 🛠️ Tech Stack (Current Baseline)

- **Framework:** Next.js 16 (App Router) + React 19
- **Language:** TypeScript
- **Database & Auth:** Supabase (PostgreSQL)
- **Styling:** Tailwind CSS v4
